# MOTD entry contract
- `date`: UTC date (`YYYY-MM-DD`); `model`: selected model name.
- `why`: `{ "en": string, "es": string }`; `sources[]`: `{ "url": string, "publishedAt": string }`.
- Selection criteria must be cited through the listed sources.
- Skipping weak days is allowed; no entry should be written without a defensible selection.
